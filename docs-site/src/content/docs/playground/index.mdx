Try the full-featured notectl editor right here in your browser. All 24 built-in plugins are active.

<Aside type="tip">
Use keyboard shortcuts like `Ctrl+B` for bold, `Ctrl+I` for italic, or type `# ` to create a heading. Use `Shift+Enter` for a hard line break. See the [Plugin Overview](/notectl/plugins/overview/) for all available shortcuts.
</Aside>

## What you see here

This playground runs the exact same `@notectl/core` package you would install in your project. The editor is configured with every built-in plugin:

| Group | Plugins |
|-------|---------|
| **Typography** | Font, Font Size |
| **Formatting** | Bold, Italic, Underline, Strikethrough, Superscript, Subscript |
| **Color** | Text Color, Highlight |
| **Structure** | Heading, Blockquote, Code Block |
| **Alignment** | Left, Center, Right, Justify |
| **Lists** | Bullet List, Numbered List, Checklist |
| **Insert** | Link, Table, Horizontal Rule, Image |
| **Direction** | Text Direction (LTR, RTL, Auto), Inline Bidi Isolation |
| **Utility** | Hard Break, Gap Cursor, Caret Navigation, Smart Paste |

The output panel below the editor shows the live document state in JSON, HTML, or plain text format.
