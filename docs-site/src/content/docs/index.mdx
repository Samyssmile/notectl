## Why notectl?

<CardGrid stagger>
	<Card title="Web Component" icon="laptop">
		Ships as a native `<notectl-editor>` custom element. Works with React, Vue, Angular, Svelte, or plain HTML — no framework lock-in.
	</Card>
	<Card title="Plugin Architecture" icon="puzzle">
		Every feature is a plugin. Text formatting, headings, tables, fonts — compose exactly the editor you need. Build your own plugins with a clean API.
	</Card>
	<Card title="Immutable State" icon="approve-check">
		All state changes go through transactions with invertible steps. Built-in undo/redo, predictable updates, and easy state inspection.
	</Card>
	<Card title="Declarative Toolbar" icon="setting">
		Configure your toolbar layout with a simple array syntax. Group plugins visually, control ordering, and get automatic separator rendering.
	</Card>
</CardGrid>
